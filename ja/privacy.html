プライバシーポリシー

最終更新日：2026年2月4日

はじめに

このプライバシーポリシーは、MeetReady（「当社」）が macOS アプリケーションをご利用いただく際に、お客様の情報をどのように収集、使用、共有するかを説明するものです。

収集する情報

Google アカウント情報

Google アカウントを MeetReady に接続すると、以下の情報にアクセスします：

  • カレンダーデータ：会議のタイトル、時間、説明、場所を含む Google カレンダーのイベント
  • 会議情報：Google Meet の会議リンクと詳細
  • 参加者情報：リアルタイム参加者追跡を可能にするための会議参加者の名前とメールアドレス

使用データ

アプリを改善するために、以下を含む分析および使用情報を収集します：

  • アプリの使用統計と機能のインタラクション
  • クラッシュレポートとエラーログ
  • デバイス情報（macOS バージョン、アプリバージョン）

情報の使用方法

収集した情報は以下の目的で使用します：

  • メニューバーに今後の会議を表示
  • 今後の会議の通知を送信
  • Google Meet 通話へのワンクリック参加を実現
  • リアルタイムの参加者情報を表示
  • アプリのパフォーマンスとユーザーエクスペリエンスの向上
  • 技術的な問題の診断と修正

データの保存とセキュリティ

カレンダーと会議のデータは Mac のローカルに保存されます。当社はお客様の情報を保護するために適切なセキュリティ対策を実施しています：

  • Google サービスとのすべての通信は安全な HTTPS 接続を使用
  • 認証トークンは macOS キーチェーンに安全に保存
  • カレンダーデータは今後の会議を表示する目的でのみローカルにキャッシュ

データの共有

当社はお客様の個人情報を第三者に販売、取引、または貸与しません。以下の場合にのみ情報を共有することがあります：

  • Google との共有：お客様の明示的な許可のもと、Google クラウドサービスを使用してカレンダーと会議のデータにアクセスします
  • 分析サービス：匿名化された使用データを分析プロバイダーと共有し、アプリの改善に役立てることがあります
  • 法的要件：法律で義務付けられている場合、または当社の権利を保護するために必要な場合

Google API サービスユーザーデータポリシー

MeetReady の Google API から受信した情報の使用および他のアプリへの転送は、限定使用要件を含む Google API サービスユーザーデータポリシーに準拠します。

お客様の権利と選択肢

お客様のデータに関して以下の権利があります：

  • アクセス：Google アカウント設定を通じて、アプリがアクセスするすべてのデータを確認できます
  • アクセスの取り消し：Google アカウント設定またはアプリ内から、いつでも MeetReady と Google アカウントの接続を解除できます
  • データの削除：アプリをアンインストールすると、ローカルに保存されたすべてのデータが削除されます

お子様のプライバシー

MeetReady は 13 歳未満のお子様を対象としていません。当社は 13 歳未満のお子様から故意に情報を収集することはありません。

プライバシーポリシーの変更

当社はこのプライバシーポリシーを随時更新することがあります。変更がある場合は、このポリシーの上部にある「最終更新日」を更新してお知らせします。
